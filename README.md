dpub annotation
===============

Documents produced by the Annotation task force of the [Digital Publishing Interest Group](http://www.w3.org/dpub/IG) of the W3C.

See also a [paged view](http://w3c.github.io/dpub-annotation/) of the document served in HTML.

If you are member of the interest group, and you wish to contribute to the content of this repo, please contact Ivan Herman (<ivan@w3.org>) or Thierry Michel (<tmichel@w3.org>), giving them your github login.
